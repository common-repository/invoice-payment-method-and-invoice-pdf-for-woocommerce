== Short Description ==

Allow your customers to select invoice payment method during checkout.Get invoice PDF during checkout with invoice payment option. 

== Features ==

*Create Invoice Payment Method Option Bases on
*Rule Base
*User Role.
*Country Base.
*Customer Base.
*Min and Max Quantity.
*Specific Product.
*Specific Category.
*Specific Tag.
